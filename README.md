# OpenInCursor
:open_file_folder: Finder toolbar app to open current folder in Cursor

Installation and usage:
- Download app or build from source
- Move Open in Cursor.app to /Applications
- Go to /Applications
- While holding Command key, drag Open in Cursor.app to Finder toolbar
- Right-click Open in Cursor.app and click Open. When the security dialog prompt appears, click Open.
- Once Cursor new window is opened, close it.
- Navigate to a folder you want to open with Cursor
- Click on the toolbar icon
- Folder will open with Cursor
